# SVG - Zoom, Pan, Pinch

> Zoom, pan, pinch an SVG with `React`, `React Spring` and `@use-gesture`.

## Getting Started

```sh
git clone git@github.com:playolaizq/svg-zoom-pan-pinch.git

cd svg-zoom-pan-pinch

npm install

npm run dev
```

See http://localhost:5173/
